Holistic counselling and wellness for corporate Melbourne. Evidence-based breathwork, sound bath meditation, energy healing and soul-led coaching that regulates nervous systems.

## Website Details
- **Company**: Stillness Reset Co.
- **Location**: Melbourne, Victoria, Australia
- **Services**: Corporate Wellness Retreats, Lunch & Reset, Sound Baths, Breathwork, Soul-Led Coaching, Women's Circles

## Development Notes
This is a statically exported Next.js site using the App Router. Modifications to structural HTML or React payloads in the build output (`.txt` files) should be done carefully to avoid breaking client-side hydration.
